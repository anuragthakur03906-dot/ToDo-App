# Todo App (React + Vite)

A simple and clean Todo application built using **React + Vite**.  
This project demonstrates basic state management and CRUD-like operations in frontend.

---

## Features

- Add new todo
- Mark todo as completed
- Delete todo
- Instant UI update (React state)
- Clean and simple UI

---

## Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- Tailwind CSS
